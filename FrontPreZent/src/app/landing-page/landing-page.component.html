PreZent

Foi destinado para auxiliar no controle de presença estudantil

Tecnologias Usadas

Backend - C#

Para o backend, utilizamos C#, pois um dos membros da equipe já tinha familiaridade com a linguagem, além do grande acervo de bibliotecas e recursos disponíveis para facilitar o desenvolvimento.

Banco de Dados - Supabase

Para o banco de dados, usamos SQL como base e o hospedamos gratuitamente no Supabase, uma plataforma de banco de dados que oferece um plano gratuito e recursos poderosos.

Frontend - Angular

Decidimos utilizar o Angular principalmente por sua estrutura simples e bem didática. Além disso, ele utiliza TypeScript, o que facilita muito o desenvolvimento. O Angular também conta com um compilador que otimiza o código, melhorando o desempenho.
